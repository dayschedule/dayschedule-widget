DaySchedule Popup Widget

A popup widget for appointment scheduling to embed a day schedule tool to open on button click

See documentation

Basic Popup

daySchedule.initPopupWidget({
  url: 'https://meet.dayschedule.com'
});

Color example

daySchedule.initPopupWidget({
        url: 'https://meet.dayschedule.com',
        color: {
          primary: 'red',
          secondary: 'pink'
        }
      });

Event page

daySchedule.initPopupWidget({
        url: 'https://meet.dayschedule.com/product-demo'
      });

Date selected

daySchedule.initPopupWidget({
        url: 'https://meet.dayschedule.com/product-demo?date=2023-10-21'
      });

Pre-fill questions

  daySchedule.initPopupWidget({
        url: 'https://meet.dayschedule.com/product-demo',
        questions: {
          name: 'Vikash',
          email: 'Vikash.rathee@dayschedule.com'
        }
      });

Hide event details

  daySchedule.initPopupWidget({
        url: 'https://meet.dayschedule.com/product-demo',
        hideEvent: true,
      });

Hide header

  daySchedule.initPopupWidget({
        url: 'https://meet.dayschedule.com/product-demo',
        hideHeader: true
      });
